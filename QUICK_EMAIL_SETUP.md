## Setup Steps

### 1. Install Resend (if not already installed)

```bash
npm install resend
```

### 2. Get Your Resend API Key

1. Go to [resend.com](https://resend.com) and create a free account
2. Navigate to **API Keys** in the dashboard
3. Click **Create API Key** and copy it

### 3. Configure Environment Variables

Add these to your `.env.local` file (create it if it doesn't exist):

```env
# Resend API Key (get from resend.com)
RESEND_API_KEY=re_your_api_key_here

# Email sender (use this for testing, or your verified domain)
EMAIL_FROM=onboarding@resend.dev

# Your app URL (optional, for email links)
NEXT_PUBLIC_APP_URL=https://yourdomain.com

# OpenAI API Key (for GPT-4o summaries)
OPENAI_API_KEY=your_openai_api_key_here
```

### 4. Configure Your Email in Dashboard Settings

1. Log into your dashboard at `http://localhost:3000` (or your production URL)
2. Click on **Settings** (gear icon)
3. Find the **Contact Email** field
4. Enter: `INFO@CLASSYTRAVEL.CA`
5. Click **Save**

### 5. Test It!

Make a test call to your ElevenLabs agent. You should receive an email at `INFO@CLASSYTRAVEL.CA` with:
- The caller's name
- The phone number
- An AI-generated summary of the conversation

## Troubleshooting

### Not Receiving Emails?

1. **Check Resend API Key**: Make sure `RESEND_API_KEY` is set in `.env.local`
2. **Restart Server**: After adding environment variables, restart your Next.js server
3. **Check Dashboard Settings**: Verify `INFO@CLASSYTRAVEL.CA` is saved in Settings > Contact Email
4. **Check Server Logs**: Look for email-related success/error messages in your terminal

### Using a Custom Domain

For production, verify your domain with Resend:

1. Go to **Domains** in Resend dashboard
2. Add your domain (e.g., `classytravel.ca`)
3. Add the DNS records they provide
4. Update `.env.local`:
   ```env
   EMAIL_FROM=notifications@classytravel.ca
   ```
